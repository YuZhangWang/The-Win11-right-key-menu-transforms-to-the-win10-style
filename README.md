# The Win11 right-key menu transforms to the win10 style
## 使用方法：

1. 右键单击「右键菜单样式.cmd」文件，然后使用管理员身份运行。 

2. 选择“2”可以切换到“Windows 10 ”的经典菜单，然后自动重启系统生效。

   测试时间：2021-10-06 
   运行效果：成功！
